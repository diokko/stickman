Function move() :

@parameters can be : "right","left","down"
Returns : nothing
Description : Execute the action of moving your stickman in the @parameter direction
Frequency : can be called up to 3 times in a turn

Function look()

@parameters : nothing
Returns : element list on the current case.
Description : look on the stickman case, searching for weapons, potions, opponent, etc...
Frequency : can be only called one time in a turn

Function protect()

@parameters : nothing
Returns : nothing
Description : protect your stickman in case of danger
Frequency : can be only called one time in a turn

Function attack()

@parameters : int idStickman
Returns : nothing
Description : Attack the stickman marked by the idStickman parameter. Each attack bring 1 damage point (iDamage)
Frequency : can be only called one time in a turn

Function grab()

@parameters : int idItem
Returns : nothing
Description : grab weapon or potion located in your Stickman case. Only 1 object is grabbed. object is added to your weapons or potions Stickman list.
Frequency : can be only called one time in a turn

Function use()

@parameters : int idItem
Returns : nothing
Description : use an object defined by it idItem passed in parameter of the function. The object used may be 1 weapon or 1 potion. Weapons are always available in your weapon list. Potions are usable one time to restore your life points
Frequency : can be only called one time in a turn

Function getPos()

@parameters : nothing
Returns : x,y
Description : return the current position of your stickman in x and y
